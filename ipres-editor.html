IPRES is a simple HTML wrapper / presentation editor (https://github.com/zglu/ipres).
To edit: click 'Add Slide' and insert your HTML code between section and /section. You can use markdown to html converter or WYSIWYG html editor to quickly get your content in html. To preview: click 'Preview' and navigate the slides using and . To export: click 'Get Code' and save the code to an .html file. To customise: add/change a css style file and/or fit it into a js presentation framework (e.g., revealjs, deckjs, webslides). (Demo)
